하지만, 우리가 자바스크립트에서 만들던 객체에 약간의 제약이 추가되어 있습니다. JSON에는 객체, 배열, 문자열, 숫자, 불, null만 들어갈 수 있습니다. 또한 문자열은 무조건 큰 따옴표를 사용해야 합니다.


JSON 단점
현재 JSON 형식의 단점으로 알려진 것이 한 가지 있는데, 데이터의 양이 엄청나게 많아지면 데이터 추출 속도가 현저하게 떨어진다는 것입니다. 따라서 데이터의 양이 많다면 CSV 형식을 사용할 것을 추천합니다.
지금까지 살펴본 데이터 전송 형식을 정리하면 다음과 같습니다.

형식	장점	단점
CSV	용량이 적다	가독성이 떨어진다
XML	가독성이 좋다	용량이 크다
데이터의 양이 커지면 분석 속도가 떨어진다
JSON	용량이 적다
가독성이 좋다	데이터의 양이 커지면 분석 속도가 떨어진다
